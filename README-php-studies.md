### Code Style
+ PSR-1, PSR-2, PSR-4, PSR-12 – Foque nesses padrões primeiro; são fundamentais para qualquer projeto limpo.
+ + AVANÇADO = PSR-7, PSR-11, PSR-15 – Padrões mais avançados para HTTP messages, containers e middlewares.
